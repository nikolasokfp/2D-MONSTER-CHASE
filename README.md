# 2D-MONSTER-CHASE
Monster Chase

A thrilling 2D survival game made with Unity.
Run, dodge, and survive while dangerous monsters chase you through endless levels. Stay focused, avoid getting caught, and see how long you can survive!

Gameplay
Escape from monsters chasing you
Avoid traps and dangerous obstacles
Survive as long as possible to increase your score
The game becomes harder over time
Features
Fast-paced 2D gameplay
Increasing difficulty
Simple and smooth controls
Endless survival mechanics
Score tracking system
Controls
A / D or Arrow Keys → Move
Space → Jump
Made With
Unity
C#
Goal

Avoid the monsters, survive as long as possible, and achieve the highest score!
